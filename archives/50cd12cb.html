vue脚手架创建项目

项目搭建
vue/cli
发布日期:   2020-05-07
更新日期:   2020-05-07
文章字数:   650
阅读时长:   2 分
阅读次数:  

什么是vue/cli？它是vue官方提供的脚手架工具，它可以自动将项目所需要的环境、依赖等信息都配置好。

1.使用命令行工具创建

1.1 安装vue/cli

npm install -g @vue/cli

1.12 创建项目

项目名称尽量小写，如果有大写字幕可能会报错

vue create <projectname>

1.3 选择手动配置

使用键盘上下键切换，默认default配置不够项目的使用，通常选择 Manually select features 回车进入下一步。如果曾经创建过项目并且保存了配置可以直接使用创建项目。

1.4 选择项目需要的配置项

使用键盘上下键进行切换，空格键用来选中与取消，回车进入下一步。

1.5 选择router模式

不使用历史模式，使用hash模式，输入n。history和hash的比较

1.6 ESLint配置

选择标准配置

1.7 选择代码检查时间点

选择第一项Lint on save，在保存后就自动检验代码和格式。

1.8 询问配置保存位置

询问是否将配置放在 package.json 文件中。推荐选第一项In dedicated config files，创建新的配置文件

1.9 是否保存为预配置

询问是否将以上配置保存为预配置，就是1.3中的mypreset选项，设置为预配置以后可以更快的创建项目。如果输入y则保存预配置，否则不保存。

上述过程完成之后，回车即开始创建项目……

1.10预览项目

项目创建完毕之后，切换到文件目录，输入npm run serve 运行项目，打开 localhost：8080 预览项目

2. 使用可视化工具创建

2.1 打开可视化面板

输入 vue ui 命令，打开可视化面板。

2.2 打开Vue项目管理器

点击“创建”，选择合适的位置创建一个项目，后续步骤的选择，同命令行选项配置一致。

3. 分析Vue脚手架生成的项目结构

​ node_modules: 依赖包目录
​ public： 静态资源目录
​ src： 源码目录
​ src/assets: 资源目录
​ src/components： 组件目录
​ src/views: 视图组件目录
​ src/App.vue: 根组件
​ src/main.js: 入口js
​ src/router.js: 路由js
​ babel.config.js: babel配置文件
​ .eslintrc.js:

4. Vue脚手架自定义配置

  1. 通过package.json进行配置【不推荐】

    "vue":{
    "devServer":{
        "port":"9990",
        "open":true
    }
}

  2. 通过单独的配置文件进行配置，在项目根目录下创建 vue.config.js

    module.exports = {
    devServer:{
        port:8888,
        open:true
    }
}
